## 课程简介

MATH241 多变量微积分主要研究多元函数的微积分理论及其应用。本课程在单变量微积分的基础上，扩展到多维空间，为后续的工程和科学课程奠定数学基础。

## 课程内容

本课程主要包含以下几个核心模块：

### 1. 向量与空间几何 (Vectors and the Geometry of Space)

- 三维坐标系
- 向量及其运算
- 空间中的点、线、面
- 距离与角度

### 2. 向量函数 (Vector Functions)

- 向量值函数
- 空间曲线
- 导数与积分
- 运动学与曲率

### 3. 偏导数 (Partial Derivatives)

- 多元函数
- 偏导数与几何意义
- 梯度与方向导数
- 极值与拉格朗日乘数法

### 4. 多重积分 (Multiple Integrals)

- 二重积分
- 三重积分
- 积分变换（极坐标、柱坐标、球坐标）
- 应用

### 5. 向量微积分 (Vector Calculus)

- 向量场
- 线积分
- 格林定理
- 曲面积分
- 散度与旋度
- 斯托克斯定理与散度定理
